About Cat Collector


Cat Collector is a site built by Anthony Kelly to allow users to keep track of cats they have seen, like, or acquired. Users can make lists of cats, edit the cat information, or delete cats as they see fit. This site began as a project for the General Assembly coding bootcamp Software Engineering Immersive. It is built using Python/Django and PostgreSQL.
